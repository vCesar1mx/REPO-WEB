## API Reference (Backend)

#### Get all data

```http
  GET /getdata
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `NA/N` | **NO Required**. |

#### Get assets

```http
  GET /css/${exact_filename_in_folder}
  GET /js/${exact_filename_in_folder}
  GET /logo
```
